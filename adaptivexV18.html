ADMIN DIAGNOSTIC TERMINAL:

ขั้นตอนการสร้างตารางและติดตั้ง Script

1. การเตรียมชีต คราวด์

สร้าง คราวด์ ใหม่ และสร้างแถบ (Sheet Tabs) ทั้งหมด 5 แถบโดยตั้งชื่อให้ถูกต้องตรงกัน ดังนี้:

  • users: (code, name, level)
  • questions: (level, type, question, answer, option1, option2, option3, option4, Thai translation)
  • business_questions: (id, level, type, question, answer, Thai translation, option1, option2, option3, option4)
  • progress: (วัน เวลา, หมวดแบบฝึกหัดใด, คะแนน ที่ทำได้, ระดับที่ทำได้, รหัสผู้เรียน, หมายเหตุ)
  • sessions: (วัน เวลา เข้าระบบ, เวลาออกจาก ระบบ, รหัสผู้เรียน)

⚠️ สำคัญ: ต้องตั้งชื่อแถบชีตสะกดให้ตรงตัวพิมพ์เล็ก-ใหญ่ และแชร์สิทธิ์เป็น "ทุกคนที่มีลิงก์มีสิทธิ์อ่าน" (Anyone with the link can view)

2. การติดตั้ง Google Apps Script (GAS)

กดเมนู "ส่วนขยาย" -> "Apps Script" ในหน้า คราวด์ แล้วนำโค้ด Apps Script ที่จัดส่งให้ไปวางบันทึก แล้วกด Deploy เป็น Web App (ให้เลือกสิทธิ์ Anyone เข้าถึงได้)
